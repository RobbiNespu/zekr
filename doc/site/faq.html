Zekr 0.1.0 FAQ
Frequently Asked Questions of the Zekr project.

[General]

What is Zekr?
Zekr is an open platform for working on the Holly Quran. For more details see a list of features for Zekr 1.0 plan here. You can also find answers to some of your questions about the Holly Quran here.
What does Zekr mean?
Zekr is a name for the Holly Quran (in Quran terms). Linguistically, it means "note", "reminder" or "message". More clearly, Quran is a reminder for the human morality.
Where is the homepage?
The main homepage for zekr is http://siahe.com/zekr, which is updated with the latest builds of Zekr. There is also a secondary home page on the leading hosting service for open source projects: sourceforge.net. The url is http://zekr.sourceforge.net. It may not update as fast as siahe.com. Conventionally package naming is of the form net.sf.zekr.*.

See the page here.

Why are you working on Zekr?
There is not a really nice and standard platform for Quran related topics. There are some software which are fully commercial, others are free-commercial but no source code available. I didn't find such standard and up to date platform. If you know one, tell us.
Which license is Zekr distributed under?
For now, GNU GPL. You can find more information about licenses at release notes page.
Where can I more information?
You can contact us by following contact link on the Zekr homepage menu.

[Developer]

What is Zekr written in?
Zekr is written in Java (JDK 1.4+ compliant). It is developed in Eclipse platform (3.0 to 3.1M4).
Which GUI library is used?
Zekr is fully developed on SWT. Since it is not included in the standard Sun's JDK, you need to have swt-win32-xxxx.dll for Win32 machines or libswt-gtk-xxxx.so/libswt-pi-gtk-xxxx. for your Linux/GTK platforms. There is also a platform specific jar file on the project's lib directory named swt-xxx.jar, wher xxx can be win32 or gtk!

[Install]

I downloaded Zekr. Now, what should I do?
Please note that there are two releases for each version of zekr for any platforms. One is installer based version. If you have downloaded this file, you can easily follow the installation steps and having JRE 1.4.1+ installed on you computer, you can then run Zekr. There is also another version available as a zipped archive which is proper for IT professionals!
I've seen an XP style screenshot of Zekr, but my copy is not in XP theme. What should I do?
It is mainly about the GUI toolkit used for Zekr (SWT). Zekr supports XP theme, but you should do something manually. Get ready? OK, let's go:
Make sure where the java.exe in your path is. Some probabilities are:
  • [Windows]/system32/java.exe
  • [JAVA_HOME]/bin/java.exe
  • [Program Files]/java/j2re1.4.x/bin/java.exe
You should copy the file java.exe.manifest and also java.exe.manifest (located at [zekr]/res) near the java.exe and javaw.exe. That's all.
Note that installer will try to put manifest files at proper locations from 0.1.0 release and there is no need to manually do that.
Which languages does Zekr support?
Zekr has Language Engine for supporting multiple languages. There are corresponding XML files for each language pack (located at [zekr]/res/lang). You can write your own language pack (e.g. [zekr]/res/lang/esperanto.xml) for Zekr, and add a new node to [zekr]/res/config/zekr-config.xml, indicating the new language pack. For example if we want to write a new language pack for Esperanto language, we can make esperanto.xml near other packs, and change zekr-config.xml this way:
Before:
	<language default="fa" current="UNDEFINED">
		<pack file="farsi.xml" id="fa" name="فارسی" />
		<pack file="english.xml" id="en" name="English" />
	</language>
After:
	<language default="fa" current="sp">
		<pack file="farsi.xml" id="fa" name="فارسی" />
		<pack file="english.xml" id="en" name="English" />
		<pack file="esperanto.xml" id="sp" name="Esperanto" />
	</language>
Please note that when current is set to "UNDEFINED" (which is the default setting), Zekr tries to find the language pack equal to user's locale language (user.language property). If it fails, it will use the default language pack isntead.
For now there are only two language packs available: Farsi and English.
