Recitations

 Recitation files listed below are compatible with Zekr 0.7.5 and newer. If you are using an older version of Zekr (check your version via: Help > About), please find older recitation list on old recitation archive. You can find instructions to add new recitations here. You can find more info on Zekr recitation file format on the wiki.
Note that some recitations here may contain duplicate or missing ayas. Splitting Qur'an into aya-by-aya files is a hard and error-prone task. You can help us by reporting such errors.
