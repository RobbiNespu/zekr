Zekr is an open platform for research and development on the Holly Quran. It is a Quran based project, planned to be a universal, open source, cross-platform application to perform most of the usual refers to Quran. The main idea is to build an as generic as possible platform to be capable of having different add-ins for its tasks.

The current version, 0.2.0, is planned to search the Quran, and support multiple languages The program is written in Java (HotSportTM JVM), using Eclipse IDE, and SWT as the widget toolkit.

After another long delay... Zekr 0.2.0 beta 1 released! (9-12-05) Zekr 0.2.0 beta 1 has just been released.
The main features in this release are:
  • Search: from Zekr 0.2 you can search throught the quran fully, or on a single soora, or either just seach on your search result. The search speed is good and gives also the result statistics.
  • Change the language: although from Zekr 0.1.0 beta 1 there is a multilingual support in Zekr, bu now it is available from a menu (view - language - your language!). This beta release only has English, Farsi and an incomplete indonesian language pack which will be completed for beta 2 release.
  • [devel] Engines: the language engine is rewritten! There is a new search engine available.

After a long delay... Zekr 0.1.0 released! (13-9-05) The Zekr Project 0.1 has just been released. It is now available with a long delay after the latest beta (b2) releases.

The main features in this release are:
  • Install now: from Zekr 0.1 windows releases will be available as a NSIS installation.
  • Smaller size: previous beta releases used JSmooth as the java Win32 executable wrapper. The size of zekr.exe was about 400kb then, which is now reduced to less than 100kb. NSIS is again used as a more simpler and featureful Java exe wrapper.
  • Bug fixes: there is a number of bug fixes and small enhancements in this release.
  • SWT Based: Thanks to Eclipse.org. Zekr uses SWT 3.1 which was released with Eclipse Platform 3.1.
