Zekr is an open platform for research and development on the Holy Quran. It is a Quran based project, planned to be a universal, open source, cross-platform application to perform most of the usual refers to Quran. The main idea is to build an as generic as possible platform to be capable of having different add-ins for its tasks.

The current release, 0.3.0, is planned to support a fully customizable theme mangement and translation management system.

The program is written in Java, on Eclipse IDE, and SWT as the widget toolkit. Please refer to release notes page for more technical details.

Zekr 0.3.0 beta 2 released! (25-3-06) Less than 2 weeks after the first beta release, the second beta release is out for download.
  • Different view layouts are now supported, which eases Quran/Translation browsing very much. These are Quran-only, Translation-only, Mixed and Separate layouts.
  • Search is more convenient now, since you can see a popup translation for any search result.
  • In both Translaton-only and Mixed layouts, you can search in the view.
  • A simple Options menu is added to set general and theme-related configurations.
This release is successfully tested on JRE 1.4.2 and 5. It also works fine on Linux and Mac. A small problem in shell script of the beta 1 release for Mac machines is now fixed. Download!

After a long delay... Zekr 0.1.0 released! (13-9-05) The Zekr Project 0.1 has just been released. It is now available with a long delay after the latest beta (b2) releases.

The main features in this release are:
  • Install now: from Zekr 0.1 windows releases will be available as a NSIS installation.
  • Smaller size: previous beta releases used JSmooth as the java Win32 executable wrapper. The size of zekr.exe was about 400kb then, which is now reduced to less than 100kb. NSIS is again used as a more simpler and featureful Java exe wrapper.
  • Bug fixes: there is a number of bug fixes and small enhancements in this release.
  • SWT Based: Thanks to Eclipse.org. Zekr uses SWT 3.1 which was released with Eclipse Platform 3.1.
