Zekr release notes

Ver 0.3.0, 15-4-06

Zekr 0.3.0 has a number of major enhancements over 0.2.0 release. Some of the most important enhancements are liste below:
  • Theme management, which means one can add/remove his own theme to the Zekr.
  • Quran translations in different languages.
  • Translation management, which means one can add/remove his own translation to the Zekr.
  • Options menu for setting general options as well as theme specific key-value settings.
  • A port on Mac platform.
See changes.txt at [zekr]/doc for more detailed changes.

Ver 0.2.0, 26-1-06

Zekr 0.2.0 has a number of major enhancements over 0.1.0 release. Some of the most important enhancements are liste below:
  • Search (matching/non-matching diacritics)
  • Print
  • HTML export
  • Advanced (enhanced) navigation

Zekr root directory (say [zekr]) contains:
  • lib - third party libraries
  • dist - source and binary distribution
  • res - resources (icons, xmls, etc.)
  • doc - Zekr documentations
  • run.bat - executable script for Win32
  • zekr.exe - Win32 executable file for launching applicatiton
