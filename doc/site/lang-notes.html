Note for Language Pack Translators
This page contains a small tutorial plus some tips for language pack translators.

From Zekr 0.1.0, there is language pack support, which means that one can easily change the user interface language in his own language.

Although this is quite easy to write a language pack, there are some tips that translators should take into consideration. The following are some tips about it:

  • Specify direction; Arabic like languages (Arabic, Farsi, Urdu) are written right to left unlike Roman characters (English, French).
  • Specify your language ID and your country (e.g. "en" as language ID and "US" as the country). The latter should be written in capital letters.
  • Use UTF-8 compliant editors. Some good editors are: Notepad, SCUniPad, Frontpage, MS Word, KWrite, and etc.
  • Please check this page out periodically to see if there is a newer English (reference) language pack. Because the software is growing there are bunch of new words and sentences with a minor release. Here are some links to current language packs available on CVS. You can download your own and see if there are new words waiting for the meaning! New words are highlighted with 3 spaces after value attribute and before closing the tag. By default the English meaning is put for all language packs when a new word is added. Here is an example: 
    <text id="QUESTION" value="Question"   />
